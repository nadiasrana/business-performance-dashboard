# Executive Business Performance Dashboard

## Business Problem
Leadership needs a unified view of business performance to understand revenue trends,
profitability, and performance across products and regions.

## Dataset
Publicly available transactional sales data (Superstore-style dataset) containing
orders, customers, products, regions, revenue, profit, and discounts.

## Tools
- SQL (PostgreSQL / SQLite)
- Power BI or Tableau
- GitHub

## Data Model
Star schema:
- fact_sales
- dim_date
- dim_product
- dim_customer
- dim_geo

## Analysis
- Revenue and profit trends
- Month-over-month growth
- Product and regional performance
- Discount impact on margin
